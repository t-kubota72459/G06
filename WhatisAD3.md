# Analog Discovery 3 (AD3) 活用ガイド
## 〜デジタル回路理論を、物理的な「波形」として理解する〜

デジタル回路の講義では、「0」と「1」の論理記号を用いて回路を設計する方法を学びました。
これからの実習では、机上の理論を実際の電子部品で動作させ、物理的な電気信号として観測・検証します。

そのための中心的なツールが、この **「Analog Discovery 3 (AD3)」** です。

---

## 1. Analog Discovery 3 とは

PCとUSB接続して使用する、**オールインワンのポータブル計測器**です。
通常、実験室のベンチに並んでいる「オシロスコープ」「ファンクションジェネレータ」「ロジックアナライザ」「直流電源」などの機能を、この一台に集約しています。

大学の研究室や企業の開発現場において、回路の動作確認やデバッグ（不具合修正）を行うための標準的なツールとして広く利用されています。

### 主な搭載機能
この実習では、以下の4つの機能を頻繁に使用します。

| 機能名称 | 役割 | 実習での用途 |
| :--- | :--- | :--- |
| **Oscilloscope**<br>(オシロスコープ) | **電圧波形の観測** | 時間経過に伴う電圧の変化をグラフ化します。<br>アナログ的な信号品質やノイズを確認するために使用します。 |
| **Wavegen**<br>(波形発振器) | **信号の生成** | 正弦波や矩形波（クロック信号）などを自由に作り出し、<br>回路に入力信号として供給します。 |
| **Logic Analyzer**<br>(ロジックアナライザ) | **デジタル信号の解析** | 複数のデジタル信号（0/1）を同時に記録します。<br>タイムチャートを自動生成し、タイミング検証を行います。 |
| **Power Supplies**<br>(可変電源) | **電力の供給** | 回路を動かすための電圧（5Vや3.3Vなど）を出力します。<br>電池や外部電源の代わりとなります。 |

---

## 2. なぜ計測器を使うのか？（学習の目的）

シミュレーション上では完璧に動く回路も、現実世界では予期せぬ挙動を示すことがあります。AD3を使用することで、以下の工学的な視点を養います。

### ① 「理想」と「現実」のギャップを知る
教科書上のデジタル信号は垂直に切り替わりますが、現実の電気信号には **「立ち上がり時間（遅延）」** や **「オーバーシュート（揺れ）」** が存在します。
オシロスコープで実際の波形を見ることで、高速通信や誤動作の原因となる物理現象を理解できます。

### ② デバッグ（問題解決）能力の習得
「回路が動かない」という時、テスターで電圧を測るだけでは原因が特定できないことが多々あります。
「信号が来ているか」「タイミングはずれていないか」を波形として可視化することで、論理的なトラブルシューティングが可能になります。

### ③ エンジニアの必須スキル
波形を見ながら回路を調整するスキルは、ハードウェア開発のみならず、組み込みソフトウェア開発やシステムエンジニアリングの現場でも必須となる能力です。

---

## 3. 安全上の注意（重要）

AD3は高機能な精密機器です。以下の点に注意して取り扱ってください。

* **ショート（短絡）の禁止**
    電源ライン（赤い線/V+）と、グランド（黒い線/GND）を直接接触させないでください。内部回路が破損する恐れがあります。
* **配線の変更手順**
    回路の配線を変更する際は、必ず WaveForms 上で **「Supplies (電源)」を OFF にしてから** 行う習慣をつけてください。

## 実習のステップ

1.  PCに制御ソフトウェア「WaveForms」をインストールする。(済んでいます)
2.  AD3を接続し、基本的な入出力テスト（ループバック）を行う。
3.  実際の電子部品（LEDやセンサー）を接続し、信号を解析する。
